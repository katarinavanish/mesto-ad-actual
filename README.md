# Место — интерактивная галерея

Учебный проект ассоциированных программ Яндекса и СГУ.

## О проекте

Интерактивная веб-страница для публикации и просмотра фотографий. Пользователь может добавлять новые карточки с изображениями, ставить и снимать лайки, удалять свои карточки, а также редактировать информацию профиля (имя и описание).

**Что использовалось:**
- HTML5, CSS3
- JavaScript (ES6+)
- Адаптивная верстка (Flexbox, Grid, медиазапросы)
- Валидация форм
- Работа с API сервера Mesto
- Webpack/Vite для сборки

**Функционал:**
- Редактирование профиля
- Добавление/удаление карточек
- Лайки карточек
- Просмотр изображений в полном размере
- Просмотр статистики карточек по клику на логотип
