# Место — интерактивная галерея

Учебный проект ассоциированных программ Яндекса и СГУ.

## О проекте

Интерактивная веб-страница для публикации и просмотра фотографий. Пользователь может добавлять новые карточки с изображениями, ставить и снимать лайки, удалять свои карточки, а также редактировать информацию профиля (имя и описание).


**Что использовалось:**
- HTML5, CSS3
- JavaScript (ES6+)
- Адаптивная верстка (Flexbox, Grid, медиазапросы)
- Валидация форм
- Работа с API сервера Mesto
- Webpack/Vite для сборки

**Функционал:**
- Редактирование профиля
- Добавление/удаление карточек
- Лайки карточек
- Просмотр изображений в полном размере
- Просмотр статистики карточек по клику на логотип

## Локальный запуск

### Установка зависимостей

```bash
npm install
```

### Запуск локального сервера для разработки
```bash
npm run dev
```

После выполнения команды проект запустится в режиме разработки по адресу http://localhost:5173.

### Сборка проекта
```bash
npm run build
```

Команда собирает оптимизированную версию проекта для продакшена с помощью Vite. Собранные файлы будут помещены в папку dist. 

### Публикация на GitHub Pages
```bash
npm run deploy
```

Эта команда автоматически запускает сборку проекта и публикует содержимое папки dist в ветку gh-pages репозитория

После успешного выполнения обновленная версия становится доступна по адресу:

https://katarinavanish.github.io/mesto-ad-actual/

### Проверка сборки перед публикацией
```bash
npm run preview
```

Локально запускает собранную версию проекта из папки dist.
